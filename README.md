# FileClickNCopy
[WIP] Simples Programm zum Kopieren/Einfügen von (ausgewählten) Dateien.

## (Lern-)Ziel
Ausführbares Programm mit dazugehöriger Benutzeroberfläche, welches die spätere Nutzung vereinfachen soll.
